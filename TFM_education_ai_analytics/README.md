# TFM Education AI Analytics

Proyecto de analítica educativa con pipeline completo de datos, autoencoder, clustering y transformer.

Si llegas por el problema de TensorFlow en RTX 5080 / Blackwell, la guía actualizada está en [docs/BLACKWELL_GPU.md](/workspace/TFM_education_ai_analytics/docs/BLACKWELL_GPU.md).

Comprobación rápida:

```bash
TFM_GPU_STRICT=1 EXECUTION_DEVICE=gpu make train_transformer
```
